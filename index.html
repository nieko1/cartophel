OpenCage Data Geocoding Control for Leaflet

A Leaflet search control that uses OpenCage Data's geocoder.

To see the plugin in action, take a look at the demo or see below.

Installation

  • Clone from GitHub: git@github.com:opencagedata/leaflet-opencage-search.git
    or
  • Download a zip or tarball archive from GitHub.
    or
  • Install with Bower: $ bower install Leaflet.OpenCage.Search.
